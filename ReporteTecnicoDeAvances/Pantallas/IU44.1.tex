\subsection{IU44.1 Gestionar pagos dentales}

\subsubsection{Objetivo}{
	Mostrar una lista con toda persona que han realizado un pago de servicio en área dental. 
}
	
\subsubsection{Diseño}
	Se muestra una caja de texto que sirve como buscador, una tabla con los campos: \begin{itemize}
		\item Nombre.
		\item Curp.
		\item Número de boleta o empleado.
		\item Fecha de confirmación.
		\item Cantidad.
		\item Acciones.
	\end{itemize}
	dentro de acciones se muestra el siguiente icono 
	\raisebox{-\mydepth}{\fbox{\includegraphics[height=\myheight]{icons/archivoSigaa}}},
	, además de un botón \IUbutton{Regresar}.
