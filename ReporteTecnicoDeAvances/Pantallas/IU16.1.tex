\subsection{IU16.1 Visualizar comprobante pago}

\subsubsection{Objetivo}{
	Permite al contador(a) visualizar el comprobante de pago.
}
	
\subsubsection{Diseño}
	Se muestra el comprobante de pago que se envió a caja y un botón \IUbutton{Regresar}.
