\subsection{IU5 Login}

\subsubsection{Objetivo}{
	Pantalla de inicio que permite a los usuarios ingresar en el sistema.
}
	
\subsubsection{Diseño}
	Esta pantalla aparece al ingresar mediante la URL del sistema. Se visualizan las cajas de texto de correo y contraseña, las opciones de registro y recuperación de contraseña, además del botón \IUbutton{Login}.

\IUfig[.6]{gui/Usuario/Registro}{IU5}{Login}

%\subsubsection{Salidas}

%	Comentarios ingresados debajo de la información del componente electrónico.

\subsubsection{Entradas}
Correo de usuario y contraseña.

\subsubsection{Comandos}
\begin{itemize}
	\item \IUbutton{Login}: Realiza una busqueda del usuario y contraseña para saber que la información ingresada sea la correcta. En caso contrario se mostrará un mensaje de error.
\end{itemize}

\subsubsection{Mensajes}
	\begin{Citemize}
		\item {\bf MSG50} Campo obligatorio.
		\item {\bf MSG55} Usuario y/o contraseña incorrectos.
	\end{Citemize}
