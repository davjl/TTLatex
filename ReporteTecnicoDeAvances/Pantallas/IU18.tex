\subsection{IU18 Visualizar historial de pagos}

\subsubsection{Objetivo}{
	Permitir al usuario visualizar un historial de todos sus pagos realizados en caja.
}
	
\subsubsection{Diseño}
	Se muestra una caja de texto que sirve como buscador y una tabla con los siguientes campos:
	\begin{itemize}
		\item Fecha.
		\item Servicio.
		\item Cantidad.
		\item Acciones
	\end{itemize}
	En el apartado de acciones se utiliza el icono
	\raisebox{-\mydepth}{\fbox{\includegraphics[height=\myheight]{icons/archivoSigaa}}}.
