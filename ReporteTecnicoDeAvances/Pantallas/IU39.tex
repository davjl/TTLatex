\subsection{IU39 Gestionar pagos caja}

\subsubsection{Objetivo}{
	Mostrar una lista con toda persona que han realizado un pago de servicio en área CELEX. 
}
	
\subsubsection{Diseño}
	Se muestra una caja de texto que sirve como buscador, una tabla con los campos: \begin{itemize}
		\item Nombre.
		\item Curp.
		\item Número de boleta o empleado.
		\item Fecha de confirmación.
		\item Cantidad.
		\item Acciones.
	\end{itemize}
	dentro de acciones se muestra el siguiente icono 
	\raisebox{-\mydepth}{\fbox{\includegraphics[height=\myheight]{icons/archivoSigaa}}},
	, además de los botones \IUbutton{Imprimir todos} y \IUbutton{Generar Excel}.

\IUfig[.6]{gui/Cajero/GestionarPagos}{IU39}{Gestionar pagos caja}

\subsubsection{Comandos}
\begin{itemize}
	\item 
	\raisebox{-\mydepth}{\fbox{\includegraphics[height=\myheight]{icons/archivoSigaa}}}: Redirecciona a la pantalla \IUref{IU32}{Visualizar comprobante sigaa}.
	\item \IUbutton{Imprimir todos}: Imprime el archivo de todos los pagos.
	\item \IUbutton{Generar Excel}: Crea un documento en excel con la información de los pagos.
\end{itemize}
