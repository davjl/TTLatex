\subsection{IU15 Gestionar pagos usuario}

\subsubsection{Objetivo}{
	Mostrar los pagos que estan en un estado de revisión por parte de caja.
}
	
\subsubsection{Diseño}
	Se muestra una caja de texto que sirve como buscador, una tabla con los campos:
	\begin{itemize}
		\item Estatus de caja.
		\item Fecha de entrega.
		\item Cantidad.
		\item Concepto.
		\item Acciones
	\end{itemize}
	Dentro de acciones se muestran los siguientes iconos
	\raisebox{-\mydepth}{\fbox{\includegraphics[height=\myheight]{icons/verPago}}}, para el estatus de caja se utiliza un semáforo de colores para identeficar el estado y fuera de la tabla se da una descripción del significado de cada color.
